## CineData

App to search information about movies and shows. I worked on it to be responsive, visually modern and fast.

### ✨ Technologies

- Typescript
- React
- Vite
- Axios
- Redux
- Tailwind
- Shadcn

API used for data: https://www.omdbapi.com/
